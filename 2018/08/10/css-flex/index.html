Flex 布局

Flex 布局？

Flex 是 Flexible Box 的缩写，意为“弹性布局”。

它的特点就是简单、快速和灵活。
它跟 grid 最大的区别是，它是一维的，而 grid 是二维的。所以 flex 的更加适合应用程序组件和小规模的布局；大规模的整体性布局的话，grid 更加适合。

基本概念

首先我们要知道Flex是有容器和容器中的项目之分的；看下面这张图：

  • 使用flex布局的元素，即为flex容器（flex container），容器里的子元素即为容器的项目（flex item）
  • 容器默认有两根轴：水平的主轴（main axis）和垂直的交叉轴（cross axis）。默认方向都是从左到右，从上到下。当然这里的主轴不一定就是水平的，交叉轴也不一定就是垂直的，这取决于 flex-direction 属性（下面会说到）
  • flex item占据的空间：占据主轴的叫 main size，占据交叉轴的叫 corss size
  • 注意：使用flex容器内元素，即flex item的float、clear、vertical-align属性将失效
