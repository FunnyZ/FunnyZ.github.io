HTML 语义化

发表于 2017-09-05 | 分类于 HTML

HTML 语义化已经是老生常谈了，在面试的时候也是经常会遇到。

那么什么是HTML 语义化呢？它又有什么作用呢？它背后的神秘力量又是谁呢？首先引用下通俗的说法：

HTML 语义化，简单说就是让机器可以读懂内容，让协作者也能迅速读懂内容。

HTML 语义化，主要指标签、属性及属性值的语义化，即用正确的标签放置正确的内容，让内容结构化。因此，约定成俗后，利于浏览器搜索引擎的解析、SEO 以及后续的阅读维护。
