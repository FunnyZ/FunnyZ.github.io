CSS 盒子模型

发表于 2017-09-08 | 分类于 CSS

概述

它是元素大小的呈现方式，由 content （内容）、padding（内边距）、border（边框）和 margin（外边距）组成。

计算方式

CSS3 新增了一个属性：box-sizing 属性，可以改变盒子模型的计算方式。

box-sizing 默认值是 content-box，它表示计算方式是标准的盒子模型计算方式；除了标准盒子模型外还有模拟 IE 低版本的非标准盒子模型计算方式 border-box。

content-box

  • 例子

  • 解析

.example-box 里的 width 只是内容的宽度, height 也只是内容的高度。浏览器渲染的整个盒子占的宽度是: 

100（ width ） + 10（ border-left ）+ 10（ border-right ） + 20（ padding-left ） + 20（ padding-right ）+ 40（ margin-left ） + 40（ margin-right ） = 240px

高度同理也是：

100（ height ） + 10（ border-top ）+ 10（ border-bottom ） + 20（ padding-top ） + 20（ padding-bottom ）+ 40（ margin-top ） + 40（ margin-bottom ） = 240px

border-box

  • 例子

  • 解析

设置 box-sizing: border-box; 其盒子模型为非标准盒子模型或 IE 盒子模型。.example-box 里的 width 包括内容的宽度、边框和内边距，同理 height 包括内容的高度、边框和内边距，其盒子模型占的宽度：

100（ width ） + 20（ margin-left ） + 20（ margin-right ） = 140px

同理，其盒子模型占的高度：

100（ height ） + 20（ margin-top ） + 20（ margin-bottom ） = 140px

说在后头

盒子模型的 4 种组成，内容、边框、内边距和外边距都有各自的值，其中 margin（即外边距）还可以是负值、在行内元素之间和块级元素之间的呈现方式也是不一样，这些探讨以后会详细记录，多做 demo。
